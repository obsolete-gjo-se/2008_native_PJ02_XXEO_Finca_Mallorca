JavaScript

 

Für die Steuerung unserer Seiten und die richtige Darstellung in vielen Bereichen der gesamten Homepage, insbesondere der Foto-Show, verwendet Fairway to Mallorca sogenannte Java Scripts. Das sind eigene kleine Programme, die innerhalb einer Homepage ausgeführt werden. Um unsere Seiten nun richtig darzustellen und den kompletten Funktionsumfang nutzen zu können, müßen Sie deshalb in Ihrem Browser die Funktion "JavaScript" aktiviert haben, was standardmäßig schon so ist, d.h., wenn Sie an Ihren Einstellungen nichts verändert haben, funktioniert die Seite einwandfrei. Falls Sie Probleme haben, die Foto-Show zu öffnen, kontrollieren können Sie das folgendermaßen:
Im Internet Explorer finden Sie diese Einstellungen unter Extras / Internetoptionen / Sicherheit. Klicken Sie bei "benutzerdefiniert" auf "Stufe anpassen". Hier muss die Option "Active Scripting" aktiviert sein.
Im Netscape finden Sie diese Einstellungen unter Bearbeiten/Einstellungen/Erweitert.

Unsere Scripts werden nur auf unserem Webserver ausgeführt, nicht auf Ihrem PC! Sie müssen also auf unseren Seiten keine Angst vor Viren oder ähnlichen Unannehmlichkeiten haben!

Darstellung

 

Das Design unserer Seiten ist für eine Bildschirmauflösung von 1024 x 768 Pixel optimiert. Wenn Sie diese Auflösung benutzen, brauchen Sie nur bezüglich der unterschiedlichen Seitenhöhen scrollen. Wenn Sie eine kleinere Auflösung benutzen, empfehlen wir, diese zu ändern:
Klicken Sie im Windows mit der rechten Maustaste auf den Hintergrund (Desktop) und wählen Sie "Eigenschaften". Klicken Sie nun auf die Registrierkarte "Einstellungen". Unten rechts in diesem Feld finden Sie den Schieberegler, mit dem Sie die Auflösung ändern können. Wählen Sie mindestens 1024 x 768 und bestätigen Sie die folgenden Felder. Aber auch an die Anwender mit der Auflösung 800 x 600 haben wir gedacht. Sie brauchen nur in den seltensten Fällen scrollen. Eine kleinere Auflösung wird nicht unterstützt.
Navigation

 

Um innerhalb unserer Seiten zu navigieren, finde Sie jeweils den Link "Home" auf jeder einzelnen Seite ganz oben in der Leiste links. Da der Internet-Auftritt von Fairway to Mallorca auf Frames basiert (das sind jeweils einzelne autarke Internet-Seiten, die in "ein großes gemeinsames" Fenster geladen werden) muss Ihr Browser Frames unterstützen. Heutzutage können das fast alle gängigen Browser.

Suchfunktionen

 

Damit Sie leichter zu Ihrem Traum-Urlaubsdomizil finden, haben wir für Sie eine Suchfunktion in unsere Seite eingebunden.
Auf der Seite Objektübersicht können Sie auf der linken Seite eine Suche entdecken.

Geben Sie z.B. den Begriff Meerblick ein und Sie erhalten in einem kleinen Fenster alle Objekte angezeigt, die über einen Meerblick verfügen. Denkbar ist auch eine Verknüfpung: Meerblick acht Fewo liefert Ihnen das gewünschte Suchergebniss über Ferienwohnungen, die mindestens über Platz für acht Personnen verfügen und einen Meerblick haben.
